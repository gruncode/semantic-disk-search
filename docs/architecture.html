semantic-disk-search

Hybrid semantic search for a local document corpus. A natural-language query returns a ranked, cited AI answer in ~4 minutes.

Entry scripts — dsearch (3 modes), dsearch-multimodel
Claude steps — query expand (1), agent ranking (4 ×10), final answer (6)
Cohere — embed-multilingual-v3.0 → ChromaDB retrieval
Extraction — PDF · DOCX · XLSX · OCR sidecars
Data stores — ChromaDB (~1M chunks), 5 FAISS indexes, Xapian BM25
Outputs — terminal list · HTML table · cited answer

Key design: agents rank files 0–10 without answering. Claude only reads the top 15 originals for synthesis. Separates recall from generation — prevents citation hallucination.

Cost: Cohere API only (~free tier for queries). Claude runs via your existing subscription — no per-token billing. Deep mode: a few extra Cohere calls.
